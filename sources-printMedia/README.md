# 📄 Fontes e Recursos de Impressão

Este diretório contém os arquivos de fontes utilizados nos materiais gráficos e documentos do projeto, desenvolvidos com o software profissional de design **Affinity**.

## 🎨 Software Utilizado

**[Affinity](https://www.affinity.studio/pt_br/download)**

## 🔤 Fontes do Projeto

As seguintes fontes foram utilizadas nos arquivos deste diretório:

| Fonte | Estilo | Uso Principal |
| ------- | -------- | --------------- |
| **Courier New** | Monoespaçada, clássica | Textos técnicos, código, máquinas de escrever |
| **Bradley Hand** | Caligráfica, cursiva | Títulos elegantes, convites, assinaturas |
| **EB Garamond** | Serifada, renascentista | Corpo de texto, literatura, documentos formais |
| **Edward Script ITC** | Caligráfica, sofisticada | Títulos decorativos, materiais premium |
| **Arial Narrow** | Sem serifa, condensada | Subtítulos, legendas, economia de espaço |
| **Times New Roman** | Serifada, clássica | Textos acadêmicos, jornalismo, documentação |
| **Big Caslon** | Serifada, histórica | Títulos impactantes, materiais vintage |
| **Chalkduster** | Informal, marcador de quadro | Títulos descontraídos, estilo juvenil |
| **Academy Engraved LET** | Serifada, entalhada | Títulos clássicos, diplomas, certificados |
| **Noto Sans NKo** | Sem serifa, internacional | Textos em línguas africanas, suporte Unicode |
